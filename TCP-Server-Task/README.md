# TCP Server & Client in C#

Simple TCP server and client implementation for learning network programming.

## Projects:
- **TcpServer** - Async TCP server that accepts messages
- **TcpClient** - Console client for testing the server

## Features:
- ✅ Async connection handling
- ✅ Echo responses
- ✅ Multiple client support
- ✅ Graceful shutdown

### Server:

cd TcpServer
dotnet run
